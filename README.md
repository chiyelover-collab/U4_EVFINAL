Nombre del proyecto: Sportclub 
Como instalar las dependencias: 
- pnpm install react-router-dom
- pnpm install bootstrap
- pnpm install react-bootstrap bootstrap sweetalert2
- pnpm install react-bootstrap bootstrap 

Tecnologias utlizadas: 
- React-bootstrap
- React-router
- Node.js
- Sweetalert2
- Javascript
- CSS
- HTML
- REACT
- Vite 

Como ejecutar el frontend: 
- pnpm run dev (en carpeta del proyecto)

Como ejecutar el backend:
- pnpm run dev (en carpeta del backend)
